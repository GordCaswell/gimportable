9.8.  Compose

Revision History
Revision $Revision: 1897 $ 2006-08-01 scb

With the Compose command, you can re-compose an image that has been decomposed into its color components. You can find more information about using this command in the Compose filter section.

9.8.1.  Activating the Command

  • You can access this command from the image menubar through Image → Mode → Compose.
