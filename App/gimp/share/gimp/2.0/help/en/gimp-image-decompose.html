9.7.  Decompose

Revision History
Revision $Revision: 2117 $ 2006-08-01 scb

With the Decompose command, you can decompose an image into its color components. You can find more information about using this command in the Decompose filter section.

9.7.1.  Activating the Command

  • You can access this command from the image menubar through Image → Mode → Decompose.
