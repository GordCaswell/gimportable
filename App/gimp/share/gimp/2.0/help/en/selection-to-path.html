7.17.  To Path

Revision History
Revision $Revision: 1993 $ 2006-08-15 scb

The To Path command converts a selection into a path. The image does not seem to change, but you can see the new path in the Paths Dialog. By using the Path tool in the Toolbox, you can precisely adapt the outline of the selection. You can find further information regarding paths in the Paths dialog section.

7.17.1.  Activating the Command

  • You can access this command from the image menubar through Select → To Path.

  • You can also access it from the Selection Editor or from the Paths Dialog which offers you a lot of Advanced Options.
